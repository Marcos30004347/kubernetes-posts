# Kubernetes Extension Server

### Description

This repository contain a custom extension server for Kubernetes, it allows you to program your own server that will be deployed in your cluter given you full flexibility and controll over its operations, it can be used to extend Kubernetes capabilityes in unlimited ways.

This server was written for my series of blog posts on medium about low level kubernetes programming. The first post is available at https://medium.com/@marcos30004347/programming-a-kubernetes-extension-api-server-part-1-60526e141a10
